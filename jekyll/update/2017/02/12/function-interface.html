其他函数接口

Interface         Description
Predicate         T->boolean
Consumer         T->void
Function         T->R
Supplier         ()->T
UnaryOperator         T->T
BinaryOperator         (T,T)->T
BiPredicate<L,R>         (L,R)->boolean
BiConsumer<T,U>         (T,U)->void
BiFunction(T,U,R)         (T,U)->R
